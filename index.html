Nix Language

Noted:

Values in the Nix language can be primitive data types, lists, attribute sets, and functions.

Nix 语言中的值可以是基元数据类型、列表、属性集和函数。

Attribute sets and let expressions are used to assign names to values. Assignments are denoted by a single equal sign (=).

属性集和 let 表达式用于为值指定名称。赋值由单个等号 （=） 表示。

Whenever you encounter an equal sign (=) in Nix language code:

每当您在 Nix 语言代码中遇到等号 （=） 时：

On its left is the assigned name.

它的左边是分配的名称。

On its right is the value, delimited by a semicolon (;).

它的右边是值，用分号 （;) 分隔。

Data Types

string

list

Attribute Set

An attribute set is a collection of name-value-pairs, where names must be unique.

属性集是名称-值-对的集合，其中名称必须是唯一的。

Attribute access

Attributes in a set are accessed with a dot (.) and the attribute name.

集合中的属性使用点 （.） 和属性名称进行访问。

Accessing nested attributes works the same way.

访问嵌套属性的工作方式相同。

The dot (.) notation can als

o be used for assigning attributes.点 （.） 表示法也可用于分配属性。

Language Constructs

Recursive attribute set

rec { … }

You will sometimes see attribute sets declared with rec prepended. This allows access to attributes from within the set.
您有时会看到以 rec 为前缀的声明属性集。这允许从集合中访问属性。

Elements in an attribute set can be declared in any order, and are ordered on evaluation.

属性集中的元素可以按任意顺序声明，并在计算时进行排序。

Let-expressions

let expressions allow assigning names to values for repeated use.

let 表达式允许为值指定名称以供重复使用。

Only expressions within the let expression itself can access the newly declared names. We say: the bindings have local scope.

只有 let 表达式本身中的表达式才能访问新声明的名称。我们说：绑定具有局部范围。

with …; …

The with expression allows access to attributes without repeatedly referencing their attribute set.with

表达式允许访问属性，而无需重复引用其属性集。

Attributes made available through with are only in scope of the expression following the semicolon (;).通过 with 提供的属性仅在分号 （;) 后面的表达式的作用域内。

inherit …

inherit is shorthand for assigning the value of a name from an existing scope to the same name in a nested scope. It is for convenience to avoid repeating the same name multiple times.inherit

是将现有作用域中的名称值分配给嵌套作用域中的相同名称的简写。为了方便起见，避免多次重复相同的名称。

String interpolation

${ … }

The value of a Nix expression can be inserted into a character string with the dollar-sign and braces (${ }).

Nix 表达式的值可以插入到带有美元符号和大括号 （${ }） 的字符串中。

File system paths

Absolute paths always start with a slash (/).

绝对路径始终以斜杠 （/） 开头。

Paths are relative when they contain at least one slash (/) but do not start with one. They evaluate to the path relative to the file containing the expression.

当路径包含至少一个斜杠 （/） 但不以斜杠 （/） 开头时，路径是相对的。它们的计算结果为相对于包含表达式的文件的路径。

以下示例假定包含的 Nix 文件位于 /current/directory 中（或者 nix repl 在 /current/directory 中运行）。

One dot (.) denotes the current directory within the given path.

一个点 （.） 表示给定路径中的当前目录。

You will often see the following expression, which specifies a Nix file’s directory.

您经常会看到以下表达式，它指定了 Nix 文件的目录。

Two dots (..) denote the parent directory.

两个点 （..） 表示父目录。

金融与周期

Posted on 1970-01-01 In Learning , Finance

1. 土地金融-危机的规律

危机，你会发现，其产生的规律，大体是相同的：
工业化、城市化产生的是真实的住房需求，并不一定就会催生泡沫。说到底，泡沫本质是杠杆。
杠杆越深，泡沫越大，也越脆弱，随便一点风吹草动都会引发崩溃。
而走向崩溃的转折点，在于资金的脱实向虚。
由于房地产是成本之王，房价泡沫会推高生活及生产成本，压缩实体利润，资金的逐利属性决定着它们一定会抽离实体去追逐金融资产，这就会导致失业率上升。
居民收入下降，消费就会下降，又让实体愈发萧条，经济进入衰退的死亡螺旋。
杠杆越来越不平衡，泡沫也越来越脆弱…
所以经济泡沫的破灭是必然这是资本主义的本质所决定的，他们就是要赚尽最后一颗铜板，不惜连绞死自己的绳子，都会亲手制造出来！
地产不光是成本之王，更是周期之母，严重的房地产危机，不仅会导致银行业的系统性危机，更会演化为社会危机。
每次危机发生之前，人们总倾向于相信这次和以往不同，认为这次是真正的繁荣，以往的教训已经被 100% 吸收。
经济学家卡门 M-莱因哈特和罗格夫写了一本书，叫《这次不一样》，讲的是 800 年的历史长河中，发生过无数次金融危机，每次人们都一厢情愿地认为「这次不一样」，但事实上，每次都有相似相同的地方。
